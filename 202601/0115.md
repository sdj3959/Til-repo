## 0115 정보처리기사 필기 (2/N): 1과목 - 네트워크 프로토콜과 신기술

---

### ✅ 1. TCP/IP 프로토콜 스위트 (Suite)

*   OSI 7계층 모델은 통신을 이해하기 위한 **참조 모델**이고, 현재 인터넷에서 **실제로 사용되는** 프로토콜의 집합은 **TCP/IP 모델(또는 TCP/IP 4계층)**입니다.

| TCP/IP 4계층 | OSI 7계층과의 관계 | 주요 프로토콜 |
| :--- | :--- | :--- |
| **응용 계층**<br>(Application) | 응용, 표현, 세션 계층 | **HTTP**, **FTP**, **SMTP**, **DNS**, SSH |
| **전송 계층**<br>(Transport) | 전송 계층 | **TCP**, **UDP** |
| **인터넷 계층**<br>(Internet) | 네트워크 계층 | **IP**, ICMP, ARP |
| **네트워크 액세스 계층**<br>(Network Access) | 데이터 링크, 물리 계층 | Ethernet, Wi-Fi |

#### ➕ 1-1. 전송 계층: TCP vs UDP (매우 중요)

*   전송 계층은 송신자와 수신자를 연결하는 **종단 간(End-to-end) 데이터 전송**을 담당합니다.

| 구분 | TCP (Transmission Control Protocol) | UDP (User Datagram Protocol) |
| :--- | :--- | :--- |
| **연결 방식** | **연결 지향 (Connection-oriented)** | **비연결 지향 (Connectionless)** |
| **특징** | • **3-way handshake**를 통해 연결을 설정.<br>• 데이터의 **전송 순서를 보장**.<br>• 데이터가 누락되면 **재전송**을 요청. | • 연결 설정 없이 데이터를 일방적으로 전송.<br>• 순서 보장 X, 재전송 X.<br>• 헤더가 단순하여 **속도가 빠름**. |
| **신뢰성** | **높음** (신뢰할 수 있는 프로토콜) | **낮음** (최선형 서비스) |
| **주요 사용 서비스** | HTTP, FTP, SMTP (신뢰성이 중요한 서비스) | DNS, VoIP, 실시간 스트리밍 (속도가 중요한 서비스) |

#### ➕ 1-2. 인터넷 계층: IP (Internet Protocol)

*   **IP**는 **패킷(Packet)**이라는 데이터 전송 단위에 **출발지와 목적지의 논리적 주소(IP 주소)**를 붙여, 네트워크상의 **최적의 경로를 찾아(Routing)** 데이터를 전송하는 역할을 합니다.
*   **특징**: 비신뢰성, 비연결성. 데이터 전송의 신뢰성은 상위 계층인 TCP가 담당합니다.

### ✅ 2. 주요 응용 계층 프로토콜

*   **HTTP (HyperText Transfer Protocol)**: 웹에서 HTML 문서와 같은 하이퍼텍스트를 주고받기 위한 프로토콜. (포트: 80)
*   **HTTPS (HTTP Secure)**: HTTP 통신을 SSL/TLS로 암호화한 보안 버전. (포트: 443)
*   **FTP (File Transfer Protocol)**: 파일 전송을 위한 프로토콜. (포트: 20, 21)
*   **SMTP (Simple Mail Transfer Protocol)**: 이메일을 **보낼 때** 사용하는 프로토콜. (포트: 25)
*   **POP3 / IMAP**: 이메일을 **받을 때** 사용하는 프로토콜.
*   **DNS (Domain Name System)**: 도메인 이름(`www.google.com`)을 IP 주소로 변환해주는 프로토콜. (포트: 53)

### ✅ 3. 정보 통신 신기술 동향

*   정보처리기사 시험에는 최신 IT 기술 용어에 대한 문제가 자주 출제됩니다.

| 기술 용어 | 설명 |
| :--- | :--- |
| **IoT (Internet of Things, 사물 인터넷)** | 세상의 모든 사물에 센서와 통신 기능을 내장하여 인터넷에 연결하는 기술. |
| **Cloud Computing** | 인터넷을 통해 서버, 스토리지, 소프트웨어 등 IT 리소스를 빌려 쓰고 사용한 만큼 비용을 지불하는 서비스. |
| **Big Data** | 기존의 데이터 처리 방법으로는 처리하기 어려운, 방대한 양의 정형/비정형 데이터 집합. |
| **AI (Artificial Intelligence, 인공지능)** | 인간의 학습, 추론, 지각 능력 등을 컴퓨터 프로그램으로 실현하는 기술. |
| **Machine Learning** | AI의 한 분야로, 컴퓨터가 데이터를 통해 스스로 학습하고 예측하는 기술. |
| **Deep Learning** | 머신러닝의 한 분야로, 인간의 뇌 신경망을 모방한 심층 신경망(DNN)을 통해 학습하는 기술. |
| **Blockchain** | 데이터를 분산된 여러 컴퓨터에 복제 및 저장하는 **분산 원장 기술**. 데이터의 위변조가 매우 어려움. |
| **NFT (Non-Fungible Token, 대체 불가능 토큰)** | 블록체인 기술을 이용하여 디지털 자산(이미지, 영상 등)에 고유한 소유권을 부여하는 기술. |
| **Metaverse** | 현실 세계와 같은 사회, 경제, 문화 활동이 이루어지는 3차원 가상 세계. |
| **5G (5th Generation Mobile Comm.)** | 4G(LTE)보다 빠른 속도(초고속), 더 많은 연결(초연결), 짧은 지연 시간(초저지연)을 특징으로 하는 5세대 이동통신 기술. |

### 📌 요약

*   인터넷은 **TCP/IP 4계층 모델**을 기반으로 동작합니다.
*   전송 계층의 **TCP**는 **신뢰성**을, **UDP**는 **속도**를 우선시하는 프로토콜이며, 이 둘의 차이점을 아는 것이 매우 중요합니다.
*   **IP**는 데이터에 주소를 붙여 최적의 경로로 전송하는 **길잡이** 역할을 합니다.
*   **HTTP, DNS, FTP** 등은 우리가 일상적으로 사용하는 응용 계층의 프로토콜입니다.
*   **IoT, 클라우드, 빅데이터, AI, 블록체인** 등 최신 IT 기술 용어의 기본 개념을 숙지해야 합니다.
