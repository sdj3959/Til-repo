# 📘 자바 복습 (0628)

## 1. 자바 기초 (0623)

### ✅ 변수와 자료형
- 기본 자료형: `int`, `double`, `boolean`, `char`
- 참조 자료형: `String`, 배열, 클래스 등

### ✅ 형 변환
- 묵시적 형 변환: 작은 타입 → 큰 타입
- 명시적 형 변환: 큰 타입 → 작은 타입 (캐스팅 필요)

### ✅ 배열
- 선언과 초기화: `int[] arr = new int[5];`
- 순회 방법: `for`, `for-each`, `while`

### ✅ 입출력
- `Scanner` 클래스 사용: `nextLine()`, `nextInt()` 등

---

## 2. 배열 응용 실습 (0624)

### ✅ 기능 구현 예시
- 배열 삽입, 삭제, 수정, 탐색, 복사 등의 기능 구현
- 배열 길이 확장을 위해 `copy()` 메서드 작성

### ✅ 2차원 배열
- 선언: `int[][] arr = new int[3][3];`
- 중첩 반복문으로 순회

---

## 3. 메서드와 클래스 기초 (0625)

### ✅ 메서드 정의
- 리턴값 존재 여부에 따라 `void` 또는 특정 자료형 사용
- 가변 인자 메서드: `int... nums`

### ✅ 클래스 설계
- 필드, 생성자, 메서드의 구조 학습
- `Car`, `Dog`, `Calculator` 등의 예제 클래스
