## 2. 회원관리 시스템 (0627)

### ✅ 주요 클래스
- `Member`: 회원 정보 관리
- `MemberRepository`: 회원 배열 저장소
- `MemberController`: 사용자 입력 처리
- `Run`: 프로그램 실행 시작점

### ✅ 핵심 기능
- CRUD 기능 구현 (등록, 조회, 수정, 삭제)
- 이메일 중복 검사 및 입력 유효성 검사 포함
- `UUID`로 고유 id 생성, `LocalDateTime`으로 가입일 저장

### ✅ 설계 포인트
- 객체지향 구조 기반으로 클래스 분리
- 배열 조작과 입력 처리 흐름 익히기
