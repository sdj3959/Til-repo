## 0518 웹 마스터 실전 (2/N): 워드프레스 설치와 초기 설정

---

### ✅ 1. 워드프레스 설치를 위한 준비물 (서버 환경)

*   워드프레스는 PHP 언어로 만들어졌고, 데이터를 MariaDB/MySQL 데이터베이스에 저장합니다. 따라서 워드프레스를 실행하기 위해서는 다음과 같은 서버 환경이 미리 구축되어 있어야 합니다. 이를 보통 **LAMP** 또는 **LEMP** 스택이라고 부릅니다.

*   **필수 구성 요소**:
    1.  **서버 운영체제**: **L**inux
    2.  **웹 서버**: **A**pache 또는 **N**ginx (**E**ngine-x)
    3.  **데이터베이스**: **M**ariaDB 또는 **M**ySQL
    4.  **프로그래밍 언어**: **P**HP

*   **사전 작업**:
    *   서버에 위 스택을 모두 설치합니다.
    *   워드프레스가 사용할 **전용 데이터베이스**와 **사용자 계정**을 미리 생성해 둡니다.

---

### ✅ 2. 워드프레스 수동 설치 과정

*   호스팅 업체가 제공하는 자동 설치 기능을 사용할 수도 있지만, 웹 마스터는 수동 설치 과정을 이해하여 문제 발생 시 대처할 수 있어야 합니다.

1.  **워드프레스 코어 파일 다운로드**:
    *   공식 사이트(`wordpress.org`)에서 최신 버전의 워드프레스 파일을 다운로드합니다. (`latest.zip` 또는 `latest.tar.gz`)

2.  **서버에 파일 업로드 및 압축 해제**:
    *   다운로드한 파일을 서버의 **웹 루트 디렉토리(Document Root)**에 업로드합니다. (e.g., `/var/www/html/`)
    *   서버에서 압축을 해제합니다. (`unzip`, `tar -xzvf`)

3.  **`wp-config.php` 파일 설정 (가장 중요)**:
    *   **역할**: 워드프레스와 데이터베이스를 연결하는 **핵심 설정 파일**.
    *   압축 해제된 파일 중 `wp-config-sample.php` 파일의 이름을 `wp-config.php`로 복사/변경합니다.
    *   `wp-config.php` 파일을 열어, 사전에 생성해 둔 데이터베이스 정보를 정확하게 입력합니다.
        ```php
        // ** Database settings - You can get this info from your web host ** //
        /** The name of the database for WordPress */
        define( 'DB_NAME', 'your_database_name' ); // DB 이름

        /** Database username */
        define( 'DB_USER', 'your_username' ); // DB 사용자명

        /** Database password */
        define( 'DB_PASSWORD', 'your_password' ); // DB 비밀번호

        /** Database hostname */
        define( 'DB_HOST', 'localhost' ); // DB 호스트 (보통 localhost)
        ```
    *   **보안 키 설정**: 파일 하단의 `Authentication Unique Keys and Salts` 부분에 있는 8개의 키 값을 공식 키 생성기 사이트에서 발급받은 새로운 값으로 교체합니다. 이는 로그인 정보의 보안을 강화하는 중요한 역할을 합니다.

---

### ✅ 3. 웹 브라우저를 통한 초기 설정 (5분 설치)

*   위의 파일 및 DB 설정이 완료되면, 웹 브라우저에서 내 도메인 주소로 접속하여 나머지 설치를 진행합니다.

1.  **사이트 정보 입력**:
    *   **사이트 제목**: 웹사이트의 이름.
    *   **사용자 이름**: 관리자(admin) 계정의 아이디. **보안을 위해 `admin`은 절대 사용하지 않습니다.**
    *   **비밀번호**: 관리자 계정의 강력한 비밀번호.
    *   **이메일 주소**: 관리자 알림을 받을 이메일 주소.

2.  **워드프레스 설치**: "워드프레스 설치하기" 버튼을 클릭하면, 워드프레스가 데이터베이스에 필요한 테이블들을 자동으로 생성하고 설치를 완료합니다.

---

### ✅ 4. 설치 후 웹 마스터의 필수 초기 작업

*   워드프레스 설치는 끝이 아니라 시작입니다. 안정적인 운영을 위해 다음과 같은 초기 작업을 반드시 수행해야 합니다.

1.  **고유 주소 (Permalinks) 설정**:
    *   **경로**: `설정 > 고유 주소`
    *   기본 설정인 `Plain` (`?p=123`)은 SEO에 매우 불리합니다. **"글 이름(Post name)"** (`/sample-post/`) 또는 "사용자 정의 구조"로 변경하여, URL이 의미 있는 단어로 구성되도록 설정합니다.

2.  **기본 콘텐츠 삭제**:
    *   "Hello world!"라는 샘플 글, 샘플 페이지, 샘플 댓글을 모두 삭제하여 깨끗한 상태에서 시작합니다.

3.  **시간대 및 언어 설정**:
    *   **경로**: `설정 > 일반`
    *   사이트의 시간대를 "서울"로, 언어를 "한국어"로 정확하게 설정합니다. (예약 발행 등 시간에 영향을 줌)

4.  **필수 플러그인 설치**:
    *   **보안 플러그인**: 악성코드 스캔, 로그인 보안 강화를 위해 필수. (e.g., Wordfence, iThemes Security)
    *   **백업 플러그인**: **가장 중요.** 파일과 데이터베이스를 정기적으로 자동 백업. (e.g., UpdraftPlus)
    *   **SEO 플러그인**: 검색 엔진 최적화를 돕는 필수 도구. (e.g., Yoast SEO, Rank Math)
    *   **캐싱 플러그인**: 사이트 로딩 속도를 향상시키기 위해 설치. (e.g., WP Super Cache, W3 Total Cache)

---

### 📌 요약

*   워드프레스를 설치하기 위해서는 **LAMP/LEMP 스택**과 **전용 데이터베이스**가 준비되어야 합니다.
*   **`wp-config.php`** 파일은 워드프레스와 데이터베이스를 연결하는 **"심장"**과 같은 역할을 하며, 정확한 DB 정보와 보안 키를 설정해야 합니다.
*   설치 직후, 웹 마스터는 **고유 주소 변경, 기본 콘텐츠 삭제, 시간대 설정, 필수 플러그인(보안, 백업, SEO, 캐싱) 설치**와 같은 초기 작업을 반드시 수행하여 안정적인 운영의 기반을 다져야 합니다.```
