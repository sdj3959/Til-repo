## 0523 웹 마스터 실전 (7/N): SEO 관리와 검색엔진 노출 최적화

---

### ✅ 1. SEO란 무엇인가?

*   **SEO(Search Engine Optimization)**는 검색엔진 최적화를 의미합니다.
*   사용자가 Google, Naver, Bing 같은 검색엔진에서 특정 키워드를 검색했을 때, 내 웹사이트가 더 잘 노출되도록 사이트 구조와 콘텐츠를 개선하는 작업입니다.
*   SEO는 단순히 검색 결과 상위에 뜨는 것만을 의미하지 않습니다. 검색엔진이 내 사이트를 **잘 발견하고, 잘 이해하고, 신뢰할 수 있게 평가하도록 만드는 작업**입니다.

#### ➕ 1-1. SEO가 중요한 이유

*   **무료 유입 증가**
    *   광고비를 쓰지 않아도 검색을 통해 사용자가 사이트에 방문할 수 있습니다.
    *   특히 브랜드명, 상품명, 이벤트명 검색 유입은 구매나 상담 전환 가능성이 높습니다.

*   **사이트 신뢰도 향상**
    *   검색 결과에 잘 노출되는 사이트는 사용자에게 더 신뢰감 있게 보일 수 있습니다.
    *   제목, 설명, 대표 이미지가 잘 정리되어 있으면 클릭률도 높아집니다.

*   **광고 효율 보완**
    *   광고는 비용을 쓰는 동안만 유입이 발생하지만, SEO는 잘 관리하면 장기적인 자산이 됩니다.
    *   광고 랜딩 페이지도 SEO 구조가 잘 잡혀 있으면 사용자 경험이 좋아집니다.

### ✅ 2. 검색엔진이 웹사이트를 이해하는 과정

*   검색엔진은 웹사이트를 무작정 노출하는 것이 아니라, 일정한 과정을 거쳐 페이지를 수집하고 평가합니다.

#### ➕ 2-1. 크롤링 (Crawling)

*   검색엔진 봇이 웹사이트의 페이지를 방문하여 HTML, 링크, 이미지, 콘텐츠 등을 확인하는 과정입니다.
*   대표적인 검색엔진 봇:
    *   Googlebot
    *   Naver Search Advisor Bot
    *   Bingbot

*   **크롤링이 잘 되려면**
    *   페이지가 외부 또는 내부 링크로 연결되어 있어야 합니다.
    *   서버가 정상적으로 응답해야 합니다.
    *   `robots.txt`에서 차단되어 있지 않아야 합니다.
    *   너무 느린 페이지나 오류 페이지가 많으면 크롤링 효율이 떨어질 수 있습니다.

#### ➕ 2-2. 인덱싱 (Indexing)

*   검색엔진이 수집한 페이지 내용을 분석하고 검색 결과에 보여줄 수 있도록 저장하는 과정입니다.
*   크롤링되었다고 해서 반드시 검색 결과에 노출되는 것은 아닙니다.
*   검색엔진이 페이지 품질이 낮거나 중복 콘텐츠라고 판단하면 인덱싱하지 않을 수 있습니다.

#### ➕ 2-3. 랭킹 (Ranking)

*   사용자가 검색어를 입력했을 때 어떤 페이지를 먼저 보여줄지 결정하는 과정입니다.
*   검색엔진은 콘텐츠 품질, 검색 의도와의 관련성, 페이지 속도, 모바일 친화성, 사이트 신뢰도 등을 종합적으로 판단합니다.

### ✅ 3. 기본 SEO 태그 관리

*   웹 마스터는 각 페이지의 HTML 메타 태그를 올바르게 관리해야 합니다.
*   특히 React, Next.js, NestJS 기반 사이트에서는 페이지별 제목과 설명이 제대로 적용되는지 확인하는 것이 중요합니다.

#### ➕ 3-1. title 태그

*   `title` 태그는 검색 결과에서 페이지 제목으로 표시될 수 있는 중요한 요소입니다.
*   사용자가 검색 결과에서 가장 먼저 보는 정보이기 때문에 명확하고 클릭하고 싶게 작성해야 합니다.

```html
<title>갤럭시 S25 사전예약 | 투게더몰 LG U+ 공식 온라인몰</title>
