- push, issue, PR, wiki 등 전반적인 권한 보유

### 📄 README.md

### 🔂 Git Fork

## 🗓️ 0515 - Rebase, Cherry-pick, Tag

### 🔁 Git Rebase
- 커밋 히스토리를 재정렬하여 선형으로 만듦

### 🧠 Interactive Rebase
- `reword`, `squash`, `drop` 등 사용 가능
- 커밋 정리용

### 🍒 Cherry-pick
- 특정 커밋만 골라 다른 브랜치에 복사

### 🔖 Tag & 버전관리
- `git tag`, `git push origin --tags`

### ⚙️ Git Alias
- 자주 쓰는 명령어를 짧게 설정

### 🛡️ 민감 정보 관리
- `.gitignore` 사전 설정 중요

### 🧭 Git Reflog
- HEAD 이동 기록 추적 및 복구

## 🗓️ 0516 - HTML 기본 구조와 작성법

### 💡 작성 팁
- 들여쓰기, 주석 활용, 의미 있는 파일명

### 🧱 제목 & 단락 태그
- `<h1>` ~ `<h6>`, `<p>`, `<br>`, `<hr>`

### 📋 목록 태그
- `<ul>`, `<ol>`, `<li>` 구조화

### 🔗 하이퍼링크
- `<a href="">링크</a>`, `target="_blank"`

### 🖼️ 이미지 & 미디어
- `<img src alt>`, `<audio>`, `<video>`

### 📊 표 태그
- `<table>`, `<tr>`, `<td>`, `<th>`

### 📝 폼 요소
- `<form>`과 `action`, `method`, `enctype` 등 속성
